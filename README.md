# Recipes CRUD API

A complete CRUD API for managing recipes built using Node.js, Express.js and MongoDB following MVC architecture.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman

## Features
- Create recipe
- Get all recipes
- Get recipe by ID
- Update recipe
- Delete recipe

## Project Structure
- models
- controllers
- routes
- server.js

## Installation

1. Clone repository
2. Run npm install
3. Create .env file

MONGO_URI=mongodb+srv://chetansabili_db_user:Offshore2001@recipes-cluster.qxhflj8.mongodb.net/?appName=recipes-cluster
PORT=8000

4. Run server
node server.js

## API Endpoints

POST /recipes  
GET /recipes  
GET /recipes/:id  
PUT /recipes/:id  
DELETE /recipes/:id  

## Deployment
Server deployed on Render.
